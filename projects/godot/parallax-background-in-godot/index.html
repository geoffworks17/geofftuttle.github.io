Parallax Background

Inspired by games like Luftrausers and Jet Lancer, I wanted to try my hand at making a sky background with parallax motion of the clouds.

Demo

Click in the window below and use arrow keys to move around: My Template

I started by creating a blue gradient background in GIMP that approximated the sky getting lighter towards the horizon.

This was set as a sprite of the first child ParallaxLayer of a ParallaxBackground. The Motion Scale of this node was left at the default value of 1.

I then added the next child layer, another ParallaxLayer with several child Sprites. Each Sprite was a copy of a cloud I got from OpenGameArt.org. The sprites were spaced manually across the background gradient.

The Motion Scale of this node was set to 1.5, which will make it move 1.5 times relative to the background layer.

I then duplicated the first cloud layer, rearranged the cloud sprites, and set the scale to 2 and 2.5. This will cause the clouds to move faster relative to the background, giving the appearance of parallax motion.

To control the scrolling of the viewport, I attached the following script to the ParallaxBackground root node: 

extends ParallaxBackground

# Option to fine tune the camera speed at runtime
export var camera_speed = 2 
 
func _process(delta: float) -> void:
	# Creates a velocity vector based on keyboard input
	var velocity = Vector2()
	if Input.is_action_pressed("ui_right"):
		velocity.x -=1
	if Input.is_action_pressed("ui_left"):
		velocity.x +=1
	if Input.is_action_pressed("ui_up"):
		velocity.y +=1
	if Input.is_action_pressed("ui_down"):
		velocity.y -=1
	if velocity.length() > 0:
		velocity = velocity.normalized()
		
	# Converts velocity vector to a pixel offset, and adds it to the scroll
	var new_offset: Vector2 = get_scroll_offset() + velocity * camera_speed
	new_offset.x = clamp(new_offset.x, -768, 0)
	new_offset.y = clamp(new_offset.y, -480, 0)
	set_scroll_offset( new_offset )
